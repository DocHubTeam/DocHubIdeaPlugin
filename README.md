# DocHub IDEA Plugin

Плагин оптимизирует работу с кодом архитектуры расширяя функциональность 
[DocHub](https://github.com/RabotaRu/DocHub/tree/archops-conception-v2).

# Функциональность

* Подсказки по структуре архитектурных манифестов
